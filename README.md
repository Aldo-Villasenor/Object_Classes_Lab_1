# Bank Account

## Data Dictionary
- std::string accountName
- double initialBal
- std::string accountHolderName
- double balance
- std::string getAccountHolderName()
- double getBalance()
- void setAccountHolderName
- void deposit
- bool withdraw

## Description
This program is used to simulate basic banking operations for multiple accounts. 
The program includes the option to add and view multiple accounts and how much 
their total balance is. 

## Features
- BankAccount class
- Private member variables such as accountHolderName and balance
- Uses public Accessor and Mutator methods
- Includes a menu-driven interface to access accounts and their balances
